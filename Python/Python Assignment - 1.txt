2. What is the difference between string and variable?

ans.  * A string is a sequence of characters enclosed in single (' ') or double (" ") quotes, used to represent textual data.
      * A variable is a name that refers to a value stored in memory. It can hold various types of data, including strings, numbers, and more

3. Describe three different data types.

ans. * Integer: Represents whole numbers, like 5 or -12.
     * Float: Represents decimal numbers, like 3.14 or -0.5.
     * String: Represents text, enclosed in quotes, like 'hello' or "world".

4. What is an expression made up of? What do all expressions do?

ans. An expression is a combination of values, variables, and operators that can be evaluated to produce a value. 
     Expressions can represent calculations or transformations of data.

5. This assignment statements, like spam = 10. What is the difference between an
expression and a statement?

ans. * An expression produces a value when it's evaluated. Examples include mathematical calculations or string concatenations.

8. Why is eggs a valid variable name while 100 is invalid?

ans. Variable names cannot start with a digit. So, eggs is a valid variable name, but 100 starts with a digit and is thus invalid.
